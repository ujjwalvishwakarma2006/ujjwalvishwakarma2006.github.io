# Portfolio Website

- A minimal looking modern portfolio website.
- Comes in two themes: light and dark.

**Note:**
1. The code is completely generated by AI. Multiple iterations were made to achieve what it looks like now.
2. The website is designed primarily to look good on desktops. Although the mobile version is equally good, you may not be as appealing as the desktop version.
3. If you want to use this as your portfolio, you don't need to change the code. Just change the data in the `portfolioData.json` file.
4. The website is static, so it can be easily hosted on GitHub Pages or any static hosting service. Build the project and upload the contents of the `dist` folder to your hosting service.
